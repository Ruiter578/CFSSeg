# Buffer 最终确认实验方案（2026-07-11）

## 结论与目的

现有 `rhl_seed=1..5` 结果中，`buffer=8224` 的平均 All mIoU 为
`0.6987133800`，高于 `buffer=8208` 的 `0.6981013164`。配对均值差为
`+0.0006120636`，但五对样本的近似 95% 置信区间为
`[-0.0000302820, 0.0012544092]`，尚不足以声称综合指标上的稳定优势。

右侧扩展中，`buffer=8232, rhl_seed=2` 的 All mIoU 为 `0.6990401963`，
是 `8224` 右侧最接近的候选。现有 `8224/seed=2` 来自较早提交，而
`8232/seed=2` 和 `8224/seed=4/5` 来自当前提交，不能直接组成同版本的
配对比较。

因此只补齐最小的三组实验，不继续向 8240 以上盲目扩展。

## 实验矩阵

| buffer | rhl_seed | 用途 |
|---:|---:|---|
| 8224 | 2 | 在当前提交重跑，补齐同版本配对 |
| 8232 | 4 | 与现有 `8224/seed=4` 配对 |
| 8232 | 5 | 与现有 `8224/seed=5` 配对 |

固定控制变量：`gamma=1`、`rhl_norm=none`、`random_seed=1`、
`batch_size=32`、VOC 15-5 sequential step1、基础权重
`20260621_baseline_bs16_16_trs`、GPU 2。

## 判定规则

使用同一提交下 `rhl_seed=2,4,5` 的三对结果比较 `8224` 与 `8232`：

1. 主指标为 All mIoU 的配对均值。
2. 同时报告旧类和新类 mIoU，避免综合均值掩盖稳定性与可塑性权衡。
3. 若 `8232` 未形成一致、具有实际幅度的优势，则停止 buffer 搜索，采用
   `8224` 作为当前推荐值。
4. 不搜索 `rhl_norm`，不启用伪标签，不把 `rhl_seed` 当作待部署超参数。
