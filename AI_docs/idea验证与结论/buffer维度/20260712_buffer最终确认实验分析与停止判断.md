## 配对统计

对 `rhl_seed=2,4,5` 分别比较 `8224` 与 `8232`：

| 指标 | buffer=8224 | buffer=8232 | 8224 - 8232 |
|---|---:|---:|---:|
| 平均 All mIoU | 0.699014 | 0.698225 | +0.000789 |
| 平均旧类 mIoU | 0.784121 | 0.784790 | -0.000669 |
| 平均新类 mIoU | 0.426673 | 0.421215 | +0.005458 |

三个 seed 上，`8224` 的 All mIoU 均高于 `8232`。其结构也非常一致：

- 旧类约下降 0.00067，即约 0.067 个百分点；
- 新类约提升 0.00546，即约 0.546 个百分点；
- 新类收益大于旧类损失，因此 All mIoU 净提升约 0.079 个百分点。

All mIoU 配对差的样本标准差为 `0.000467`。由于只有三对样本，使用
t 分布得到的近似 95% 置信区间为 `[-0.000371, 0.001949]`，仍跨越零。
这表示样本量不足以作出严格的“统计显著优于”声明，但不推翻工程判断：
三对方向完全一致，而且新类提升具有稳定、可解释的幅度。

## 与 buffer=8208 的关系

此前 `rhl_seed=1..5` 的结果为：

| buffer | 平均 All mIoU | 标准差 | 平均旧类 | 平均新类 |

`8224` 相比 `8208` 同样表现为轻微牺牲旧类、改善新类，并使综合指标上升。
因此从 `8208 -> 8224 -> 8232` 的证据看，`8224` 是当前局部最优候选；
`8232` 已出现回落，继续向更大 buffer 搜索缺乏趋势依据。

## 对抗性审查

### 不能声称的内容

1. 不能声称 `8224` 是数学意义或所有数据集上的全局最优值。
2. 不能把单次最高 `0.700157` 当作可稳定复现的固定性能。
3. 不能把 `rhl_seed` 选择成“最佳 seed”后作为超参数报告；seed 是稳定性
   检查维度，不是方法改进。
4. 不能声称 All mIoU 的优势已经达到传统 0.05 水平的统计显著性。

### 可以声称的内容

1. 在本代码库、VOC 15-5 sequential step1、固定控制变量和已测试范围内，
   `8224` 是证据最强的 buffer 候选。
2. `8224` 对 `8232` 的优势在三组配对 RHL seed 上方向一致。
3. `8224` 的主要收益来自新类学习改善，而旧类仅有很小幅度下降。
4. 右侧扩展已形成回落证据，不需要继续盲目扩大 buffer。

## 是否继续启动实验

### 当前决定：不启动

不启动以下实验：

- `buffer > 8240`：现有曲线没有单调上升趋势；
- 更多 `rhl_seed`：只提高置信度，预计不会改变工程选择；
- `rhl_norm`：属于已搁置的方法变量，不应混入干净超参数搜索；
- 调整 `gamma`：当前实验协议已经按要求固定为 1；
- 伪标签参数：属于独立方法分支，与 buffer 选择不是同一问题。

如果后续目标从“选择 buffer”变为“为论文提供严格统计证据”，可以另立
复现实验：在固定 `buffer=8224` 后改变全局训练 `random_seed`，至少运行
3-5 次并报告均值和标准差。该实验验证的是完整训练流程的随机稳定性，
不再属于 buffer 搜索，因此本轮不自动启动。
