## 结论边界

第一性原理上，这些结果只比较同一 `task=15-5`、同一 setting、同一 seed、同一 batch size 32、同一 step0 复用来源下的 step1 伪标签策略。它们不能直接外推到 `15-1` 多步协议，也不能和 batch size 64 或不同 checkpoint 来源的结果混为同一 baseline。

对抗性审查后，当前较稳妥的判断是：`artifact_class` 自适应阈值在 `15-5 overlap/disjoint` 的 paired seed 复核中是稳定弱正向信号，幅度约 `+0.0003` 到 `+0.0004` all mIoU；它不是已经证明的大幅涨点方法。若该结论是错的，最可能的错误来源是：step1 单步协议过短、15-5 新类一次性进入导致伪标签筛选空间有限、或 fixed0.6 本身已经接近该机制在当前协议下的上限。最小反证实验是转到 `15-1 overlap` 多步协议，观察误差随 step1-step5 累积时自适应阈值是否放大收益或暴露退化。
